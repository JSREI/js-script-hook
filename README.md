## 一、脚本简介

为什么有这个东西？

在进行JS逆向的时候，经常碰到请求是JS类型的，实际上是使用script标签发出的请求，比如一些跨域的请求，一些jsonp请求，
Chrome目前（2024-12-20）没有支持对script类型的请求打条件断点之类的，而这个脚本就是填补这块儿的空白的。
